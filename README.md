### Project Title
Bikeshare Data Analysis 

### Description
This project contains tools for analyzing BikeShare data in an interactive way. To run the tool run from the command line:
``` 
python3 bikeshare_2.py 
```


### Files used
It uses 3 data files as inputs:
* chicago.csv
* new_york_city.csv
* washington.csv
